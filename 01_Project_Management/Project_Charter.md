# Project Charter

## Project Title

Egg Grading Machine (EGM-01)

---

## Project Overview

The Egg Grading Machine (EGM-01) is a semi-automated egg grading system designed to improve the efficiency of egg grading operations for small and medium-scale poultry farms.

The machine automatically transports eggs from a feeding tray to a weighing station using a chain-driven conveyor system. Each egg is weighed using a precision load cell, classified according to predefined weight categories, and sorted into the appropriate collection tray.

The system aims to reduce manual labour, minimize egg breakage, improve grading consistency, and increase operational productivity.

## Problem Summary

Egg grading at the client's farm is currently performed manually.

This process:

- requires significant labour

- causes frequent egg breakage

- depends heavily on worker efficiency

- produces inconsistent productivity

The client requires an affordable semi-automated machine capable of improving the grading process.

---

## Proposed Solution

Develop a semi-automated egg grading machine capable of:

- transporting eggs automatically

- measuring egg weight

- classifying eggs according to weight

- sorting eggs into separate containers

- recording grading statistics
