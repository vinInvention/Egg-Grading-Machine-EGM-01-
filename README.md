# Egg Grading Machine (EGM-01)

## Project Overview

The Egg Grading Machine (EGM-01) is a semi-automated egg grading system designed to assist small and medium-scale poultry farmers in grading eggs based on weight while minimizing manual labour, reducing egg breakage, and improving productivity.
