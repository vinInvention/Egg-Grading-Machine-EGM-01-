# Project Scope

## Project

Egg Grading Machine (EGM-01)

---

# Purpose

This document defines the boundaries of the EGM-01 project by identifying what is included and excluded from the first version of the machine. It ensures that project resources remain focused on solving the client's primary problem while preventing unnecessary expansion of project requirements.

---

# Project Goal

To design, build, test, and deploy a semi-automated egg grading machine that accurately grades eggs by weight while reducing manual labour, minimizing egg breakage, and improving operational efficiency for small and medium-scale poultry farms.

---

# Out of Scope

The following features are intentionally excluded from EGM-01.

- AI-based egg inspection
- Crack detection using computer vision
- Mobile application
- Cloud storage
- Internet connectivity
- Automatic egg feeding
- Voice control
- Multi-lane grading
- Industrial-scale production
- Automatic carton packaging

These features may be considered for future versions.

---

# Project Constraints

The project shall:

- Remain affordable for small and medium poultry farms.
- Use locally available components where practical.
- Operate without an Internet connection.
- Be serviceable using standard tools.
- Be safe for operators.
